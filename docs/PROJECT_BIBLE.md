# 📘 БИБЛИЯ ПРОЕКТА: «ИИ-КОМПАНЬОН ГЕКСЛИ»

## 1. МИССИЯ И ЦЕННОСТИ
**Проблема:** Пользователь (архетип Гексли) обладает сильными навыками ведения переговоров, креативностью и способностью убеждать. Однако «в тени» находится системная работа, монотонное отслеживание задач и дедлайнов.
**Решение:** Интеллектуальный внешний агент-секретарь, который берет на себя системность, напоминания и структурирование хаоса, позволяя пользователю фокусироваться на его сильных сторонах.

**Ценности Агента:**
*   **Четкость**: Никакой «воды», только факты, задачи и дедлайны.
*   **Партнерство**: Общение на «ты», поддержка драйва и креативности, но мягкое возвращение к реальности.
*   **Проактивность**: Агент должен сам напоминать о забытых делах, а не только ждать вопроса.

---

## 2. АРХИТЕКТУРА (STACK)
*   **Ядро**: Node.js v20+
*   **Интерфейс**: Telegram (Telegraf.js)
*   **Мозг**: Google Gemini 1.5 Flash (через Gemini API)
*   **База данных**: Google Firebase Firestore (Облачная NoSQL БД)
*   **Хостинг**: Firebase Cloud Functions (Serverless, работает 24/7)
*   **CI/CD**: GitHub Actions (Автоматический деплой при `git push`)

---

## 3. СТРУКТУРА ДАННЫХ (FIRESTORE)
*   **Коллекция `projects`**: Название, описание, дата создания, статус.
*   **Коллекция `tasks`**: Суть задачи, принадлежность к проекту, дедлайн, статус (pending/completed).
*   **Коллекция `history`**: Контекст общения (последние 20 сообщений) для поддержания осмысленной беседы.

---

## 4. ФУНКЦИОНАЛЬНЫЕ МОДУЛИ
1.  **Трекер Проектов**: Анализ входящего текста из Gemini-чатов и автоматическое создание/обновление карточек проектов.
2.  **Менеджер Задач**: Извлечение задач из контекста беседы ("напомни...", "надо сделать...", "запиши...").
3.  **Аналитический Срез** (`/status`): Ежедневный или по запросу отчет о состоянии всех дел.
4.  **Проактивные напоминания** (В РАЗРАБОТКЕ): Scheduled-функции, которые дергают пользователя, если задача просрочена.

---

## 5. ПРАВИЛА ВЗАИМОДЕЙСТВИЯ (SYSTEM PROMPT)
*   Стиль: Партнерски-деловой.
*   Обращение: На «ТЫ».
*   Реакция на креатив: «Крутая идея! Хвалю за полет мысли. Давай теперь приземлим это: какие конкретно шаги сделаем и к какому числу записать?»
